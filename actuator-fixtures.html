ROBOTIC ACTUATOR TESTING FIXTURE

Tesla

Personal project I undertook in which I created two mechatronic testing fixtures to test linear and rotary actuators. The intent is to test the actuators under extreme loading conditions to outline feasibility for world class robots such as Tesla's Optimus.

  • Designed linear and rotary actuator fixtures for life cycling testing, maximum load testing, and accelerated life tests.
  • Utilized rotary and linear actuators designs to represent Tesla's unique actuator design.
  • Used KiCAD to create electronic wiring schematic for fixture functionality.
  • Programmed script in Python to facilitate data collection/compilation and component functionality logic.
